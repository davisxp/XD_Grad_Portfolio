## Using the Portfolio

### Interactive Viewer (Recommended)

The live viewer provides a read-only, browser-based experience for exploring models:

1. **Access the Viewer** ‚Äî Click "Open Live Viewer" above
2. **Select a Workbook** ‚Äî Choose from available financial models
3. **Navigate Sheets** ‚Äî Start with "Summary" or "Valuation" sheets for overviews
4. **Explore Scenarios** ‚Äî Toggle between Bear, Central, and Bull case scenarios
5. **Inspect Details** ‚Äî Use the formula inspector to examine underlying calculations
6. **Share Views** ‚Äî Generate shareable links for specific model views

**Note:** The online viewer is read-only for inputs. Scenario switching is available, but all other modifications require downloading the Excel files.
